
## FastAPI commands to run locally:


1. To install FastAPI:

    --> pip install fastapi


2. To install uvicorn server:

    --> pip install "uvicorn[standard]"


3. To run FastAPI applicatin/project locally:

    --> uvicorn main:app --reload

    Here, main:app mean,  'from main import app'


4. To install extra model libraries/packages from requirements.txt

    --> pip install -r requirements.txt
